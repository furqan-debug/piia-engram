## 项目核心能力
1. 将用户身份、偏好、质量标准、经验教训、决策记录存为本地 JSON
2. 通过 MCP 协议分享给 Claude Code、Cursor、Codex 等 AI 工具
3. 61 个 MCP 工具（13 个 Tier-1 默认加载，48 个 Tier-2 可选）
4. engram setup 自动注入 shared instructions 到各工具配置
5. Claude Code Hooks Adapter（10+ 消息自动 wrap_up_session）
6. staging → verified 知识治理模型（AI 提议，用户批准后永久化）
7. 匿名 feedback 报告管道（opt-in，CF Worker + D1）
8. engram doctor 环境诊断
9. 安全审计修复（F-04~F-10：输入验证、路径脱敏、ISO 时间解析、tail-read audit log）
10. 与 AGENTS.md/CLAUDE.md/.cursorrules 的明确差异化定位
